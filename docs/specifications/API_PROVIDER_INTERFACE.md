## 1. Unified Interface Strategy
To achieve absolute decoupling from vendor SDKs, SurMaya enforces strict, strongly typed interface boundaries for all modal actions (Lyrics, Instrument Generation, and Vocal Synthesis). All third-party providers must implement these internal interfaces.

## 2. API Interface Definitions (Kotlin)

### 2.1 Lyrics Provider Interface
Responsible for structural song layouts, rhyming patterns, mood-based vocabulary generation, and structured metadata.

```kotlin
package com.example.domain.repository

import com.example.domain.model.Lyrics
import kotlinx.coroutines.flow.Flow

interface LyricsProvider {
    /**
     * Generates a structural song lyric set using LLM orchestration.
     * @param prompt User's creative input description
     * @param genre Music style (e.g. Ghazal, Pop, EDM)
     * @param mood Target emotional state (e.g. Melancholic, Energetic)
     * @return Result containing structured Lyrics domain model
     */
    suspend fun generateLyrics(
        prompt: String,
        genre: String,
        mood: String
    ): Result<Lyrics>

    /**
     * Streams structural song lyrics block by block for real-time preview.
     */
    fun streamLyrics(
        prompt: String,
        genre: String,
        mood: String
    ): Flow<LyricsStreamChunk>
}

data class LyricsStreamChunk(
    val section: String, // e.g., "Verse 1", "Chorus"
    val textSegment: String,
    val isComplete: Boolean
)
```

### 2.2 Audio Model Provider (Music Generator)
The engine interface that handles music rendering, instrumentation generation, and melodic rendering.

```kotlin
package com.example.domain.repository

import kotlinx.coroutines.flow.Flow

interface AudioModelProvider {
    /**
     * Generates the instrumental track.
     * @param prompt Detailed production prompts (generated by LLM)
     * @param durationSeconds Length of the generated audio track
     * @param tempo Target BPM
     * @return Result wrapping the binary AudioTrack info
     */
    suspend fun generateAudioTrack(
        prompt: String,
        durationSeconds: Int,
        tempo: Int
    ): Result<AudioTrackResult>

    /**
     * Initiates asynchronous generation, returning a polling/status token.
     */
    suspend fun initiateAsyncGeneration(
        prompt: String,
        durationSeconds: Int
    ): Result<String> // Returns Job ID
    
    suspend fun checkStatus(jobId: String): Result<GenerationProgress>
}

data class AudioTrackResult(
    val audioUrl: String,
    val audioBytes: ByteArray?,
    val durationMs: Long,
    val format: String // "wav", "mp3"
)

data class GenerationProgress(
    val jobId: String,
    val progress: Float, // 0.0 to 1.0
    val status: String, // "PENDING", "PROCESSING", "COMPLETED", "FAILED"
    val resultUrl: String?
)
```

### 2.3 Speech Synthesis & Vocalist Provider (Singer)
Responsible for taking structural text (lyrics), melody guidelines (MIDI or reference pitch tracks), and synthesizing vocal audio with pitch accuracy and natural pronunciation.

```kotlin
package com.example.domain.repository

interface VocalSynthesisProvider {
    /**
     * Synthesizes lyrical vocal lines over a target backing track or a capella.
     * @param lyrics Complete text to sing
     * @param voiceId Character/vocal model signature (e.g., "Sufi Male", "Karan")
     * @param pitchOffset Fine-tuning pitch correction (semitones)
     */
    suspend fun synthesizeVocals(
        lyrics: String,
        voiceId: String,
        pitchOffset: Float
    ): Result<VocalResult>
}

data class VocalResult(
    val vocalUrl: String,
    val audioBytes: ByteArray?,
    val voiceCharacteristics: VoiceMeta
)

data class VoiceMeta(
    val gender: String,
    val style: String,
    val languageCode: String
)
```

## 4. Error Code Taxonomy & Semantic Mapping
The Gateway interceptor translates raw HTTP status codes and library errors into structured, domain-specific `SurMayaException` models. This prevents unparsed HTTP stack traces from reaching the UI.

| HTTP Status | Raw Error | SurMaya Exception Type | Resolution Strategy |
|-------------|-----------|------------------------|---------------------|
| **400** | Invalid JSON / Schema mismatch | `ClientSchemaException` | Log details to telemetry; prompt user with simpler inputs. |
| **401** | Unauthorized / Expired Token | `InvalidApiKeyException` | Notify user immediately via a dialog leading directly to **Settings / API Config**. |
| **403** | Forbidden Resource | `PermissionDeniedException` | Fall back to standard/open models. |
| **429** | Rate Limited / Token Exhausted | `RateLimitExceededException` | Introduce local back-off timers; temporarily lock the execution button. |
| **500** | Downstream Server Failure | `VendorServerException` | Automatically activate local synth engine fallback. |
| **503** | Service Overloaded | `VendorUnavailableException` | Retry exactly once after a 3000ms delay. |
| **-1** | No connection / Host unresolved | `NoNetworkException` | Seamless transition to absolute offline database rendering. |
